DIANA OCERIN ROSANO
LAS ESTRELLAS TRANSFORMAN EL HIDROGENO EN HELIO PARA BRILLLAR.
CUANDO ESTE HIDROGENO SE ACABA Y EL NUCLEO SOLO ES DE HELIO,
LA ESTRELLA SE VUELVE MAS FRIA Y MAS BRILLANTE, HAY UNA EXPLOSION.
SE MUEREN Y SUS RESTOS SE ORIGINAN UNA ESTRELLA DE NEUTRONES.
A ESTO SE LE CONOCE COMO "SUPERNOVA"
